Для запуска через Visual Studio необходимо:
1. Запустить VS
2. Последовательность File->Open->Folder
3. Выбрать для запуска SeaBattle.exe

///Немного про проект///
Игра представляет классический морской бой, на карте случайным образом располагаются корабли:

1 четырехпалубный,
2 трехпалубных,
3 двухпалубных,
4 однопалубных.

Ходы происходят по очереди, начинает всегда игрок. Ход производится путем ввода двух координат предполагаемого выстрела,после чего на игровом поле противника отображается
результат(подробнее про обозначения расскажет сама игра). В случае попадания можно ходить еще раз, в случае промаха ход переходит к сопернику.
Побеждает тот, кто первым уничтожит корабли противника.

Изначальную идею с игрой двух человек по сети реализовать не вышло, в нынешнем формате возможна только игра игрок-компьютер.

Для реализции консольной графики используется библиотека console_graphics_master(файлы console.h/.c).
